# MyChecklist — เช็คลิสต์ส่วนตัว (ใช้คนเดียว ไม่มีเซิร์ฟเวอร์)

แอปจดงาน + เตือนถึงกำหนด ที่ **เปิดมาแล้วจดได้ทันที** — ไม่มี login ไม่มีเซิร์ฟเวอร์
ไม่ผูกกับใคร ข้อมูลทั้งหมดเก็บอยู่ **ในเครื่องของคุณ** (localStorage) และติดตั้งลงหน้าจอมือถือเป็นแอปได้ (PWA)

## ฟีเจอร์
- 🧭 **เปิดแอปเห็นงานทันที** — แถบนำทางล่าง (รายการ/ปฏิทิน/เสร็จแล้ว) ใช้นิ้วโป้งสะดวก + ปุ่ม **＋ ลอย** เปิดฟอร์มเพิ่มงานแบบ bottom sheet (เพิ่มติดกันหลายงานได้รวดเดียว)
- 🍩 **วงแหวนความคืบหน้าวันนี้** บนหัวจอ (เช่น เสร็จ 2/5) + หัวข้อกลุ่มมีตัวนับ
- เพิ่มงานเร็ว (พิมพ์ + Enter), ปุ่มวันด่วน (วันนี้/พรุ่งนี้/+7/ไม่มีกำหนด), เวลา, ทำซ้ำ (วัน/สัปดาห์/เดือน/ปี), แท็ก/อาคาร
- 📋 **มุมมองรายการ** จัดกลุ่มอัตโนมัติ: **เลยกำหนด** / **วันนี้** / **ถัดไป** / **📥 ไม่มีกำหนด (Inbox)** + แท็บ **เสร็จแล้ว**
- 📅 **มุมมองปฏิทินรายเดือน** — เห็นจำนวนงานในแต่ละวัน แตะวันเพื่อดู/เพิ่มงาน
- ☑️ **งานย่อย (subtasks)** พร้อมแถบความคืบหน้า (เช่น 2/5) — เหมาะกับงานตรวจเช็ก
- 🔥 **สถิติ** — ทำต่อเนื่องกี่วัน (streak) + เสร็จใน 7 วันล่าสุด
- ติ๊กเสร็จ → งานทำซ้ำสร้างรอบถัดไปให้เอง; แก้ไข/ลบ; **Undo** ตอนลบ/ติ๊กเสร็จ
- ⭐ **ปักหมุดงานสำคัญ** + 🔀 **จัดเรียง** (ตามวัน/สำคัญก่อน/เพิ่มล่าสุด)
- ⏰ **เลื่อนงาน** (พรุ่งนี้/+3/+7/ไม่มีกำหนด) + **เตือนล่วงหน้าต่องาน** (ตรงเวลา…1 สัปดาห์)
- ⚙️ **ตั้งค่า** — ธีม, เตือนล่วงหน้าเริ่มต้น, วันเริ่มสัปดาห์, ล้างข้อมูลทั้งหมด
- 🌙 **โหมดมืด** (ตามระบบ + เลือกเองได้) · 🧹 **ล้างงานที่เสร็จ** · 🔍 **ค้นหา** + **กรองตามแท็ก (มีสีประจำแท็ก)** · 📝 **โน้ต**
- 📋 **นำเข้ากิจกรรม** (ตั้งค่า → นำเข้ากิจกรรม): วางตารางจาก Excel/LINE คั่นด้วย Tab หรือ “|” → แปลงวันไทย พ.ศ.→ค.ศ. อัตโนมัติ, งานสอบ/ส่งงานตั้ง ⭐ + เตือนล่วงหน้าให้เอง, มีพรีวิวเลือกก่อนนำเข้า
- 👆 **ปัดนิ้วบนงาน** (มือถือ): ปัดขวา = เสร็จ · ปัดซ้าย = เลื่อนไปพรุ่งนี้ (มี Undo ทั้งคู่)
- 📆 **ส่งออกปฏิทิน (.ics)** — import เข้า Google Calendar แล้วได้การแจ้งเตือนของ Google แม้ปิดแอปสนิท (รวมงานซ้ำเป็น RRULE + เวลาเตือนล่วงหน้าต่องาน)
- 💾 **เตือนสำรองข้อมูลอัตโนมัติ** เมื่อเกิน 7 วันไม่ได้ backup + ขอ persistent storage กันเบราว์เซอร์ลบข้อมูล
- 🔄 **อัปเดตถึงเครื่องทันที** — service worker แบบ network-first + แจ้ง "มีเวอร์ชันใหม่" ให้กดรีเฟรช
- แจ้งเตือน: ในแอป + การแจ้งเตือนของระบบ + สั่นเบา ๆ ตอนติ๊กเสร็จ
- วันที่แบบไทย + ปี พ.ศ. · **สำรอง/กู้ข้อมูล** JSON · **ออฟไลน์** + ติดตั้งลงหน้าจอ (PWA)

## วิธีใช้งาน (เลือกวิธีใดก็ได้)

### วิธีที่ 1 — Netlify Drop (ง่ายสุด ได้ลิงก์ทันที ฟรี ไม่ต้องสมัครก็ได้)
1. เปิด https://app.netlify.com/drop
2. **ลากโฟลเดอร์ `mychecklist` ทั้งโฟลเดอร์** ไปวางในหน้าเว็บ
3. รอสักครู่ ได้ลิงก์ เช่น `https://xxxx.netlify.app` → เปิดใช้ได้เลย

### วิธีที่ 2 — เปิดในเครื่อง (เทสต์เร็ว)
เพราะ Service Worker/การแจ้งเตือนต้องรันผ่าน http (ไม่ใช่ `file://`) ให้เปิดเซิร์ฟเวอร์เล็ก ๆ:
```bash
cd mychecklist
python3 -m http.server 8000
# แล้วเปิด http://localhost:8000
```
> ถ้าดับเบิลคลิก `index.html` ตรง ๆ ตัวจดงานยังใช้ได้ แต่ PWA/ออฟไลน์/การเตือนของระบบจะไม่ทำงาน

### วิธีที่ 3 — Vercel (static ไม่ต้องตั้ง env)
1. Vercel → **Add New → Project** → import repo `chawanansuk/checklist`
   (import repo ตรง ๆ — **อย่าใช้ปุ่ม Deploy แบบ clone** เพราะจะสร้าง repo ใหม่)
2. **Root Directory** → กด Edit → เลือกโฟลเดอร์ **`mychecklist`**
3. Framework Preset: **Other** (ปล่อยอัตโนมัติ), ไม่ต้องใส่ Environment Variables ใด ๆ
4. **Deploy** → ได้ลิงก์ `https://<ชื่อ>.vercel.app` ใช้ได้เลย

### วิธีที่ 4 — GitHub Pages
push โฟลเดอร์นี้ขึ้น repo → Settings → Pages → เลือก branch + โฟลเดอร์ → ได้ลิงก์ `https://<user>.github.io/...`

## ติดตั้งลงมือถือ (PWA)
- **Android/Chrome:** เปิดลิงก์ → เมนู ⋮ → “ติดตั้งแอป / Add to Home screen”
- **iPhone/Safari:** เปิดลิงก์ → ปุ่มแชร์ → “เพิ่มไปยังหน้าจอโฮม”

## ข้อจำกัดของการแจ้งเตือน (อ่านสักนิด)
บนเว็บที่ไม่มีเซิร์ฟเวอร์ การเด้งเตือนตอน **ปิดแอปสนิท** ทำได้จำกัด โดยเฉพาะ **iOS**:
- **ขณะเปิดแอปอยู่:** เตือนในแอป + การแจ้งเตือนของระบบทำงานปกติ
- **ปิดแอปสนิท:** เบราว์เซอร์ทั่วไปยังไม่รองรับ → **ทางลัดที่ใช้ได้จริง: กด “📆 ส่งออกปฏิทิน” แล้ว import เข้า Google Calendar** จะได้การแจ้งเตือนของ Google เต็มรูปแบบแม้ปิดแอป
- **แนะนำ:** ติดตั้งเป็น PWA ลงหน้าจอ แล้วเปิดใช้เป็นประจำ
- ถ้าต้องการ push จริงในตัวแอปเอง ค่อยต่อ Web Push (VAPID) + เซิร์ฟเวอร์เล็ก ๆ ทีหลัง (นอกสโคปนี้)

### วิธี import .ics เข้า Google Calendar
1. กด **📆 ส่งออกปฏิทิน** ในแอป → ได้ไฟล์ `mychecklist.ics`
2. เปิด [calendar.google.com](https://calendar.google.com) → ⚙️ **Settings → Import & export**
3. เลือกไฟล์ → เลือกปฏิทินปลายทาง → **Import**
> การ import เป็นการถ่ายรูป ณ ตอนนั้น — งานใหม่ที่เพิ่มทีหลังต้อง export/import ซ้ำ

## ข้อมูล & ความปลอดภัย
- ข้อมูลอยู่ใน `localStorage` ของเบราว์เซอร์ที่คุณใช้เท่านั้น (ไม่ส่งออกไปไหน)
- **ล้างข้อมูลเบราว์เซอร์ = ข้อมูลหาย** → กด **สำรองข้อมูล** เก็บไฟล์ JSON ไว้เป็นระยะ
- ย้ายเครื่อง/เบราว์เซอร์: ใช้ **สำรอง** ที่เครื่องเก่า แล้ว **กู้ข้อมูล** ที่เครื่องใหม่

## โครงไฟล์
```
mychecklist/
  index.html          # โครงหน้า
  styles.css          # ดีไซน์
  app.js              # ตรรกะทั้งหมด (localStorage, แจ้งเตือน, ทำซ้ำ)
  sw.js               # service worker (ออฟไลน์ + คลิกการแจ้งเตือน)
  manifest.webmanifest
  icons/              # 192 / 512 / maskable
  README.md
```

ไม่มี build step, ไม่มี dependency — เป็น HTML/CSS/JS ล้วน แก้ไขได้ตรง ๆ
